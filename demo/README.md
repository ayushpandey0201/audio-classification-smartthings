# Demo Assets

This directory is intended for holding visual and auditory assets that demonstrate the project's capabilities. These are especially useful for your portfolio, GitHub landing page, or Amazon ML School application.

## Suggested Contents to Add Before Submission

1. **`demo.gif` or `demo_screenshot.png`**
   - Record a quick screen capture of your Telegram bot in action.
   - Show yourself uploading an audio file and receiving the "🍳 Cooking" response.

2. **`sample_cooking.wav`**
   - A 5-10 second clear clip of sizzling, frying, or chopping.
   - **Expected Bot Output:** Cooking (confidence > 85%)

3. **`sample_non_cooking.wav`**
   - A 5-10 second clip of generic kitchen noise (running water, microwave beeping, or talking) that should *not* trigger the cooking detection.
   - **Expected Bot Output:** Not Cooking
